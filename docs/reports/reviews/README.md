# Reviews — PR レビュー結果・コードレビュー記録

**用途**: PR レビューや `/simplify` 実行結果の記録。
Phase 12（Adversarial Review）導入後は自動生成される予定。

## 入る内容例

- 大型 PR のレビュー結果（手動）
- `/adversarial-review` の出力（Phase 12 以降）
- `/ultrareview` 結果のスナップショット

## ファイル形式

- ファイル名: `YYYY-MM-DD_pr<番号>_<slug>.md`
- PR 番号がない場合（ローカルレビュー）は `YYYY-MM-DD_local_<slug>.md`
- 新規作成時は `_registry.md` の reviews/ セクション最上部に 1 行追加
