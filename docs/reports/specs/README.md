# Specs — 仕様書（短期スパン）

**用途**: `designs/` で扱わない短期スパンの仕様書。
1 機能未満の小さな仕様、外部 API のラッパー仕様、ツール仕様等。

## 入る内容例

- ホットフィックス用の小さな仕様書
- CLI ツールの引数仕様
- データフォーマットの一時定義

## ファイル形式

- ファイル名: `YYYY-MM-DD_<slug>.md`
- 確定した仕様で長期参照されるものは `designs/` または `instruction-formats/` に移す
- 新規作成時は `_registry.md` の specs/ セクション最上部に 1 行追加
