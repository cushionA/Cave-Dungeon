# ツール一覧

## Python開発ツール

| ツール | コマンド | 用途 |
|--------|---------|------|
| feature-db | `python tools/feature-db.py` | 機能管理（SQLite） |
| generate-images | `python tools/generate-images.py` | KaggleでFLUX.2バッチ画像生成 |
| asset-index | `python tools/asset-index.py` | 音声ライブラリからLLMマッチング |
| generate-placeholders | `python tools/generate-placeholders.py` | プレースホルダーアセット生成 |
| register-assets | `python tools/register-assets.py` | アセット登録 |
| match-audio | `python tools/match-audio.py` | 音声マッチング |
| analyze-playtest | `python tools/analyze-playtest.py` | MLプレイテスト分析 |

### feature-db コマンド

```bash
python tools/feature-db.py init                    # DB初期化
python tools/feature-db.py add "名前" --tests t.cs --impl i.cs  # 機能追加
python tools/feature-db.py update "名前" --status complete --test-passed 5  # 更新
python tools/feature-db.py get "名前"               # 個別取得
python tools/feature-db.py list [--status ...]      # 一覧
python tools/feature-db.py summary                  # サマリー
python tools/feature-db.py assets                   # アセット一覧
python tools/feature-db.py add-asset                # アセット追加
python tools/feature-db.py bind                     # アセットバインド
```

## Claude Code スキル

### 設計系

| スキル | 用途 |
|--------|------|
| `/design-game` | 対話型GDD作成 |
| `/design-systems [section]` | システム設計書 + 機能分解 + feature-db登録（旧 plan-sprint 統合） |
| `/design-stage` | 2Dステージ設計 |

### 実装系

| スキル | 用途 |
|--------|------|
| `/create-feature` | 1機能TDD実装 |
| `/create-event` | イベントシーン構築 |
| `/create-ui` | UI画面構築（UI Toolkit） |
| `/create-balance-sheet` | バランスデータシート作成 |
| `/create-map-reference` | HTML参照マップ作成 |

### テスト・品質

| スキル | 用途 |
|--------|------|
| `/run-tests` | テスト実行 |
| `/simplify` | コードレビュー（品質・効率・再利用） |
| `/validate-scene` | シーン検証（MCP経由） |
| `/debug-assist` | デバッグ支援（MCP経由） |
| `/test-game-ml` | MLプレイテスト実行・分析 |

### アセット系

| スキル | 用途 |
|--------|------|
| `/generate-assets` | 画像生成 + 音声マッチング（音声インデックス管理 `index` サブコマンド内包） |
| `/bind-assets` | プレースホルダー差替え |

pending アセット一覧は `python tools/feature-db.py assets --status pending` で確認（旧 `/list-assets` は廃止）。

### その他

| スキル | 用途 |
|--------|------|
| `/build-pipeline` | 全工程オーケストレーション |
| `/adjust-stage` | ステージ調整 + 学習記録 |
| `/manage-flags` | フラグ管理 |
| `/drawio` | draw.ioダイアグラム生成 |
