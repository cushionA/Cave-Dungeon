# セクション構成

ゲーム全体を4つのセクション（開発の大きな区切り）に分割。各セクション完了時にゲームとして動作確認可能。

## 全体像

| # | セクション | 概要 | 機能数 | テスト数 | 状態 |
|---|-----------|------|--------|---------|------|
| 1 | MVP: 戦闘と探索の基盤 | 基本戦闘・装備・セーブ・マップ | 35 | ~170 | 実装済み |
| 2 | AI・仲間・連携 | 仲間AI・敵AI・魔法・連携 | 38 | ~190 | 実装済み |
| 3 | 世界の広がり | ボス・召喚・混乱・属性ゲート | 25 | ~135 | 実装済み |
| 4 | エンドコンテンツ | チャレンジモード・リーダーボード | 11 | ~41 | 実装済み |

**合計: 104機能 / 536テスト**

## Section 1: MVP — 戦闘と探索の基盤

**完了条件**: 1エリアを探索し、基本戦闘・装備変更・ジャストガードが機能する

### 含むシステム

| システム | 概要 |
|---------|------|
| DataContainer | SoAコンテナ + SourceGenerator |
| GameManager | 中央ハブ + GameEvents(R3) |
| InputSystem | Input Systemパッケージによる入力管理 |
| PlayerMovement | 移動・ジャンプ・ダッシュ + AbilityFlag |
| EquipmentSystem | 武器/盾/コア装着・ステータス再計算 |
| WeaponSystem | コンボ・チャージ・落下攻撃・飛翔体 |
| DamageSystem | 7属性ダメージ計算・状態異常・アーマー |
| ParryGuardSystem | ジャストガード・ガードブレイク |
| MapSystem | カメラ追従・ミニマップ |
| SaveSystem | ISaveable永続化・ファストトラベル |
| CurrencySystem | 通貨管理・デスペナルティ |
| ShopSystem | 購入・売却 |
| LevelUpSystem | 経験値・能力値振り分け |
| InventorySystem | アイテム管理 |

## Section 2: AI・仲間・連携

**完了条件**: 仲間AIが追従・戦闘し、連携ボタンとAIルール構築が機能する

### 含むシステム

| システム | 概要 |
|---------|------|
| AICore | AIBrain 3層判定・ConditionEvaluator・ActionExecutor |
| CompanionAI_Basic | 仲間追従・スタンス切替 |
| EnemySystem | 敵AI・DamageScoreTracker・スポーン管理 |
| AIRuleBuilder | CompanionAIConfig・AIMode・プリセット管理 |
| CoopAction | 連携ボタンスキル・コンボ対応 |
| GateSystem | Clear/Ability/Key/Elemental 4種ゲート |
| CooldownReward | CT消化済みならMP無料 |
| MagicSystem | 詠唱→発動→飛翔体・ProjectileSystem |

### ActionSlot統一

全行動をActionSlotで統一（5分類）:

| ActionExecType | 用途 |
|---------------|------|
| Attack | コンボ・ヒットボックス管理 |
| Cast | 詠唱→発動→飛翔体 |
| Instant | ワープ・回避・アイテム使用 |
| Sustained | ガード・追従・挟撃 |
| Broadcast | ターゲット指示・集合・挑発 |

## Section 3: 世界の広がり

**完了条件**: 複数エリアをボスで区切り、召喚・混乱・属性パズルが機能する

### 含むシステム

| システム | 概要 |
|---------|------|
| BossSystem | BossController + フェーズ遷移 + アリーナロック |
| SummonSystem | 召喚魔法・最大2枠・パーティ管理 |
| ConfusionMagic | 蓄積型混乱・陣営フラグ反転・最大3体 |
| ElementalGate | 属性攻撃による環境パズル |
| BacktrackReward | 能力獲得時にバックトラック報酬再評価 |

## Section 4: エンドコンテンツ

**完了条件**: チャレンジモード・スコア記録・AIテンプレートが機能する

### 含むシステム

| システム | 概要 |
|---------|------|
| ChallengeMode | ボスラッシュ・サバイバル・タイムアタック |
| Leaderboard | スコア・タイム記録・新記録判定 |
| AdvancedAITemplates | テンプレート保存・適用・推薦 |

## 依存関係

```
Section 1（基盤）
    ↓
Section 2（AI・仲間）← Section 1のシステムを利用
    ↓
Section 3（世界拡張）← Section 1-2のシステムを利用
    ↓
Section 4（エンドコンテンツ）← Section 1-3のシステムを利用
```
