# ゲーム概要

## コンセプト

| 項目 | 内容 |
|------|------|
| ジャンル | 2Dメトロイドヴァニア（HD 2D） |
| テーマ | 姉妹 |
| ターゲット | アクションゲーム好き + コアゲーマー（AIカスタムやり込み） |
| プラットフォーム | PC |
| プレイ時間 | 本編約20時間 + エンドコンテンツ |

## コアループ

```
探索 → 戦闘 → 連携発見 → 成長・カスタム → 新エリア開通 → バックトラック → 探索...
```

1. **探索**: マップを探索し、新エリア・装備・魔法・NPCを発見
2. **戦闘**: プレイヤー + 仲間AIで敵と戦闘。連携ボタンで支援魔法発動
3. **成長・カスタム**: レベルアップ、装備変更、仲間AIルール構築
4. **開通**: エリアクリアまたは連携アクションで新エリアへ
5. **バックトラック**: 新能力で過去エリアに戻り隠し要素を発見

## 面白さの核: 仲間AIカスタム

### 二重報酬構造

| 方式 | 対象 | MP消費 | 説明 |
|------|------|--------|------|
| 連携ボタン | カジュアル〜全員 | 仲間MPを消費 | ボタン1つで支援魔法を発動 |
| AIルール自動発動 | やり込み勢 | CT消化済みなら無料 | 条件ルールで最適タイミング自動発動 |

- **カジュアル層**: AIカスタムしなくても連携ボタンで十分遊べる
- **やり込み層**: 条件付きAIルールを組んでクールタイム管理を最適化
- **エンドコンテンツ**: ボスラッシュ・タイムアタックでAI最適化の真価

### AIカスタムの深さ

プレイヤーが仲間AIに条件付きルールを構築:

- **条件**: 「敵HPが30%以下」「プレイヤーHP50%以下」「特定属性の敵」等
- **行動**: 「追撃」「回復魔法」「バフ」「防御優先」等
- **優先度**: ルール間の優先順位を設定
- **モード**: 最大4モード + 自動切替条件 + ショートカット手動切替

## キャラクター構成

| 種別 | 最大数 | 説明 |
|------|--------|------|
| プレイヤー | 1 | 操作キャラクター |
| 常駐仲間 | 1 | 常にプレイヤーと行動するAI仲間 |
| 召喚 | 0〜2 | 魔法で一時召喚（合計パーティ最大4人） |
| 混乱敵 | 0〜3 | 混乱魔術で一時味方化（パーティ枠外） |

## メカニクス概要

### 戦闘
- 武器切替（最大2スロット）、コンボ分岐、チャージ攻撃
- パリィ/ジャストガード、ガードブレイク
- 7属性統一（斬撃/打撃/刺突/炎/雷/聖/闇）
- 状態異常蓄積モデル

### 成長
- レベルアップ + 能力値振り分け（STR/DEX/INT）
- 装備: 武器(右手) / 盾(左手) / コア(1スロット)
- 片手持ち(盾併用) ⇔ 両手持ち(盾無効、武器スキル使用可)
- 装備重量比率でアクション性能変動
- 魔法・能力は探索で拾得（スキルツリーではない）

### 探索
- Additive Scene Loadingによるシームレスエリア遷移
- セーブポイントで回復 + セーブ + ファストトラベル
- AbilityFlag獲得でバックトラック報酬解放
