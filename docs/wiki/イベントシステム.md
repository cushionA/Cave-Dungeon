# イベントシステム

## R3 Subject<T> パターン

GameEventsクラスで全システム間通信を管理。R3のSubject<T>を使用。

### 基本パターン

```csharp
public class GameEvents
{
    // 内部: Subject<T>
    private readonly Subject<DamageResult> _onDamageDealt = new();

    // 外部公開: Observable<T>（読み取り専用）
    public Observable<DamageResult> OnDamageDealt => _onDamageDealt;

    // 発火メソッド
    public void FireDamageDealt(DamageResult result)
    {
        _onDamageDealt.OnNext(result);
    }
}
```

### 購読パターン

```csharp
// 購読
IDisposable subscription = GameManager.Events.OnDamageDealt
    .Subscribe(result => HandleDamage(result));

// 解除（OnDisable/OnDestroyで）
subscription.Dispose();
```

## 主要イベント一覧

### Section 1（基盤）

| イベント | データ型 | 用途 |
|---------|---------|------|
| OnDamageDealt | DamageResult | ダメージ発生通知 |
| OnCharacterDeath | int (hashCode) | キャラクター死亡 |
| OnEquipmentChanged | EquipmentChangeData | 装備変更 |
| OnLevelUp | LevelUpData | レベルアップ |

### Section 2（AI・仲間）

| イベント | データ型 | 用途 |
|---------|---------|------|
| OnEnemyDefeated | int (hashCode) | 敵撃破 |
| OnCoopActionExecuted | CoopActionData | 連携アクション実行 |
| OnCustomRulesChanged | Unit | AIルール変更通知 |

### Section 3（世界拡張）

| イベント | データ型 | 用途 |
|---------|---------|------|
| OnBossDefeated | string (bossId) | ボス撃破 |
| OnGateOpened | GateOpenData | ゲート開放 |
| OnAbilityAcquired | AbilityFlag | 新能力獲得 |

### Section 4（エンドコンテンツ）

| イベント | データ型 | 用途 |
|---------|---------|------|
| OnChallengeCompleted | ChallengeResult | チャレンジ完了 |
| OnNewRecord | (string, ChallengeRank) | 新記録達成 |
| OnChallengeStateChanged | ChallengeState | チャレンジ状態変化 |

## 設計原則

- **Subject<T>はprivate**: 外部からはObservable<T>でのみ購読可能
- **Fire*()メソッドで発火**: 直接OnNextを呼ばない
- **購読は必ず解除**: IDisposableをOnDisable/OnDestroyでDispose
- **旧System.Action<T>は使わない**: 全てR3 Subjectに統一
