# 物理レイヤー定義

## レイヤー一覧

| Layer | 名前 | 用途 |
|-------|------|------|
| 0 | Default | Unity既定 |
| 1 | TransparentFX | Unity既定 |
| 2 | Ignore Raycast | Unity既定 |
| 4 | Water | 水面 |
| 5 | UI | UI |
| 6 | Ground | 地形・足場 |
| 7-9 | (予備) | 旧Player/Enemy/Companion。陣営識別はSoA `CharacterBelong`で行う |
| 10 | PlayerHitbox | プレイヤー攻撃判定（Trigger） |
| 11 | EnemyHitbox | 敵攻撃判定（Trigger） |
| 12 | CharaPassThrough | キャラクター通常状態（すり抜け） |
| 13 | CharaCollide | キャラクター衝突状態（物理衝突有効） |
| 14 | CharaInvincible | キャラクター無敵状態（すり抜け + ダメージ無効） |

## キャラクター当たり判定レイヤー（12-14）

キャラクター同士の衝突はデフォルトで**すり抜け**。アクション時に`gameObject.layer`の1行で切替（O(1)）。

### レイヤー別動作

| レイヤー | キャラ同士 | 地形 | Hitbox | 用途 |
|----------|-----------|------|--------|------|
| CharaPassThrough (12) | すり抜け | 衝突 | 検知される | 通常時 |
| CharaCollide (13) | **衝突** | 衝突 | 検知される | StopOnHit / Carry |
| CharaInvincible (14) | すり抜け | 衝突 | **検知されない** | 無敵アクション中 |

### 切替タイミング

```
通常 → CharaPassThrough (Layer 12)
  ↓ アクション開始（StopOnHit / Carry）
CharaCollide (Layer 13)
  ↓ ActionEffect.Invincible 有効
CharaInvincible (Layer 14)
  ↓ アクション終了
CharaPassThrough (Layer 12)
```

### 運搬（Carry）

`AttackContactType.Carry`のアクションがCharaCollideレイヤーで接触すると、対象が以下の状態の場合のみ運搬開始:

- **Flinch**（怯み）
- **Stunned**（スタン）
- **GuardBroken**（ガードブレイク）
- **Knockbacked**（吹き飛ばし）

かつ対象が**敵**（`CharacterBelong.Enemy`）であること。味方・中立は運搬不可。

### コード定数

```csharp
GameConstants.k_LayerCharaPassThrough = 12;
GameConstants.k_LayerCharaCollide     = 13;
GameConstants.k_LayerCharaInvincible  = 14;
```

### 設定ツール

メニュー `Game > Physics2D Collision Matrix設定` でCollision Matrixを一括設定。
