# 物理レイヤー定義

## レイヤー一覧

| Layer | 名前 | 用途 |
|-------|------|------|
| 0 | Default | Unity既定 |
| 1 | TransparentFX | Unity既定 |
| 2 | Ignore Raycast | Unity既定 |
| 4 | Water | 水面 |
| 5 | UI | UI |
| 6 | Ground | 地形・足場 |
| 7 | (予備) | 旧Player — 廃止。陣営識別はSoA CharacterBelongで行う |
| 8 | (予備) | 旧Enemy — 廃止 |
| 9 | (予備) | 旧Companion — 廃止 |
| 10 | PlayerHitbox | プレイヤー攻撃判定（Trigger） |
| 11 | EnemyHitbox | 敵攻撃判定（Trigger） |
| 12 | CharaPassThrough | キャラクター通常状態（すり抜け） |
| 13 | CharaCollide | キャラクター衝突状態（物理衝突有効） |
| 14 | CharaInvincible | キャラクター無敵状態（すり抜け＋ダメージ無効） |

## キャラクター当たり判定レイヤー（12-14）

### 設計方針
- キャラクター同士の衝突はデフォルトで**すり抜け**
- アクション（攻撃等）の`AttackContactType`に応じてレイヤーを切り替える
- `gameObject.layer`の1行で切替完了（IgnoreCollisionのループ不要）

### レイヤー別動作

| レイヤー | キャラ同士 | 地形 | Hitbox | 用途 |
|----------|-----------|------|--------|------|
| CharaPassThrough (12) | すり抜け | 衝突 | 検知される | 通常時 |
| CharaCollide (13) | 衝突 | 衝突 | 検知される | StopOnHit/Carry |
| CharaInvincible (14) | すり抜け | 衝突 | 検知されない | 無敵アクション中 |

### Collision Matrix

```
             Ground  PlayerHB  EnemyHB  CharaPT  CharaCol  CharaInv
Ground         -        -        -        O        O         O
PlayerHitbox   -        -        -        O        O         -
EnemyHitbox    -        -        -        O        O         -
CharaPT        O        O        O        -        O         -
CharaCollide   O        O        O        O        O         -
CharaInvincible O       -        -        -        -         -
```

O = 衝突有効、- = 衝突無効

### 切替タイミング

| 状況 | レイヤー | 切替元 |
|------|---------|--------|
| 通常時（Idle/Run/Jump等） | CharaPassThrough | CharacterCollisionController.Start() |
| AttackContactType.StopOnHit | CharaCollide | CharacterCollisionController.SetCollisionMode() |
| AttackContactType.Carry | CharaCollide | CharacterCollisionController.SetCollisionMode() |
| ActionEffect.Invincible有効中 | CharaInvincible | CharacterCollisionController.SetCollisionMode() |
| アクション終了 | CharaPassThrough | CharacterCollisionController.ClearCollisionMode() |
