# 04 - ダメージ・被弾システム

## 1. 概要

本システムは**与ダメージ（DamageDealer）**と**被弾（DamageReceiver）**の2つのコアコンポーネントで構成される。
攻撃判定から最終ダメージ適用まで、パリィ・ガード・多属性計算を一貫して処理する。

### 1.1 設計原則

| 原則 | 説明 |
|---|---|
| **GetComponent 禁止** | すべてのデータは `GameManager.Data`（SoACharaDataDic）に `objectHash` でアクセスする |
| **構造体ベース** | DamageData / DamageResult は struct。GC Alloc ゼロ |
| **属性別計算** | 7属性（無・火・水・風・土・光・闇）を個別に計算し、弱点倍率・物理タイプ耐性を適用 |
| **単一責任** | DamageDealer は「ダメージを作る」、DamageReceiver は「ダメージを受ける」のみ |

### 1.2 システム全体の位置づけ

```
WeaponHolder（攻撃入力・コンボ管理）
    │
    ▼ AttackInfo 参照
DamageDealer（ダメージ生成・ヒットボックス管理）
    │
    │ DamageData を生成
    ▼
DamageReceiver（被弾判定・ダメージ適用）
    │
    ├─→ パリィ判定
    ├─→ ガード判定
    ├─→ 多属性ダメージ計算
    ├─→ HP減算（GameManager.Data 経由）
    ├─→ 状態異常適用
    └─→ AIイベント通知（AIManager.NotifyEvent）
```

## 2. DamageData 構造体

攻撃者から被弾者に渡されるダメージ情報の一括パケット。

```csharp
/// <summary>
/// 攻撃時に生成されるダメージ情報パケット。
/// DamageDealer が生成し、DamageReceiver に渡す。
/// </summary>
public struct DamageData
{
    /// <summary>攻撃者の objectHash</summary>
    public int attackerHash;

    /// <summary>被弾者の objectHash</summary>
    public int defenderHash;

    /// <summary>属性別ダメージ値（7属性）</summary>
    public ElementalDamage damage;

    /// <summary>吹き飛ばし力</summary>
    public float knockbackForce;

    /// <summary>吹き飛ばし方向（正規化済み）</summary>
    public Vector2 knockbackDir;

    /// <summary>武器の物理ダメージタイプ（斬撃/打撃/刺突）</summary>
    public WeaponPhysicalType physicalType;

    /// <summary>付与する状態異常情報</summary>
    public StatusEffectInfo statusEffectInfo;

    /// <summary>状態異常の発動確率（0〜100）</summary>
    public byte statusChance;

    /// <summary>攻撃の種別（通常攻撃/スキル/魔法）</summary>
    public AttackType attackType;

    /// <summary>飛翔体による攻撃かどうか</summary>
    public bool isProjectile;
}
```

### 2.1 AttackType 列挙型

```csharp
/// <summary>
```

### 2.2 ElementalDamage 構造体

`ElementalStatus`（06_情報クラス群 参照）と同一の7属性構造で、ダメージ値専用の構造体。
物理ダメージタイプ（斬撃/打撃/刺突）は属性とは別軸で `WeaponPhysicalType` として管理する。

```csharp
/// <summary>
/// 属性別ダメージ値。7属性それぞれの最終ダメージを保持する。
/// ElementalStatus と同じフィールド構成だが、
/// 「攻撃力・防御力」ではなく「計算済みダメージ」を表す。
/// </summary>
public struct ElementalDamage
{
    public int none;      // 無属性ダメージ
    public int fire;      // 火属性ダメージ
    public int water;     // 水属性ダメージ
    public int wind;      // 風属性ダメージ
    public int earth;     // 土属性ダメージ
    public int light;     // 光属性ダメージ
    public int dark;      // 闇属性ダメージ

    /// <summary>全属性の合計ダメージ</summary>
    public int Total => none + fire + water + wind + earth + light + dark;

    /// <summary>指定属性のダメージを取得</summary>
    public int Get(Element element) => element switch
    {
        Element.None  => none,
        Element.Fire  => fire,
        Element.Water => water,
        Element.Wind  => wind,
        Element.Earth => earth,
        Element.Light => light,
        Element.Dark  => dark,
        _ => 0,
    };

    /// <summary>
    /// ElementalStatus（攻撃力/防御力）から ElementalDamage を生成する。
    /// baseDamage を各属性の攻撃力比率で按分する。
    /// </summary>
    public static ElementalDamage FromAttackStatus(
        in ElementalStatus atkStatus, int baseDamage)
    {
        int totalAtk = atkStatus.ReturnSum();
        if (totalAtk <= 0) return default;

        return new ElementalDamage
        {
            none  = baseDamage * atkStatus.none  / totalAtk,
            fire  = baseDamage * atkStatus.fire  / totalAtk,
            water = baseDamage * atkStatus.water / totalAtk,
            wind  = baseDamage * atkStatus.wind  / totalAtk,
            earth = baseDamage * atkStatus.earth / totalAtk,
            light = baseDamage * atkStatus.light / totalAtk,
            dark  = baseDamage * atkStatus.dark  / totalAtk,
        };
    }
}

/// <summary>物理ダメージタイプ（属性とは別軸）</summary>
public enum WeaponPhysicalType : byte
{
    Slash,    // 斬撃
    Strike,   // 打撃
    Pierce,   // 刺突
}

/// <summary>
/// 物理タイプ別耐性。CharacterInfo で定義し、SoAコンテナに格納する。
/// </summary>
public struct PhysicalResistance
{
    public float slash;    // 斬撃耐性（0.0=等倍, 正=軽減, 負=弱点）
    public float strike;   // 打撃耐性
    public float pierce;   // 刺突耐性

    public float Get(WeaponPhysicalType type) => type switch
    {
        WeaponPhysicalType.Slash  => slash,
        WeaponPhysicalType.Strike => strike,
        WeaponPhysicalType.Pierce => pierce,
        _ => 0f,
    };
}
```

## 3. DamageDealer（与ダメージ側）

攻撃のダメージ情報を生成し、ヒットボックスやプロジェクタイル経由で DamageReceiver に送信する。

```csharp
/// <summary>
/// 与ダメージ管理コンポーネント。
/// 攻撃情報（AttackInfo）とキャラクターの攻撃ステータスから DamageData を生成する。
/// データは全て GameManager.Data に objectHash でアクセスする。
/// </summary>
public class DamageDealer : MonoBehaviour
{
    /// <summary>所有者の objectHash（GameManager.Data へのキー）</summary>
    private int _ownerHash;

    /// <summary>現在の攻撃情報（WeaponHolder から設定される）</summary>
    private AttackInfo _currentAttackInfo;

    // ===============================================================
    //  初期化
    // ===============================================================

    /// <summary>
    /// BaseCharacter の Initialize から呼ばれる。
    /// objectHash を受け取り、以降のデータアクセスに使用する。
    /// </summary>
    public void Initialize(int ownerHash)
    {
        _ownerHash = ownerHash;
    }

    // ===============================================================
    //  ダメージ計算
    // ===============================================================

    /// <summary>
    /// 現在の AttackInfo とキャラクターの攻撃ステータスから DamageData を生成する。
    /// </summary>
    /// <param name="defenderHash">被弾者の objectHash</param>
    /// <returns>計算済みの DamageData</returns>
    public DamageData CalculateOutgoingDamage(int defenderHash)
    {
        // GameManager.Data 経由で攻撃ステータスを取得（GetComponent 不使用）
        ref CharacterAtkStatus atkStatus =
            ref GameManager.Data.GetAtkStatus(_ownerHash);

        // AttackInfo の baseDamage に攻撃力比率を掛けて属性別ダメージを算出
        ElementalDamage elemDamage = ElementalDamage.FromAttackStatus(
            in atkStatus.atk,
            _currentAttackInfo.baseDamage
        );

        // KnockbackInfo から吹き飛ばし情報を取得
        var knockback = _currentAttackInfo.knockbackInfo;

        // StatusEffectInfo から状態異常情報を取得
        var statusInfo = _currentAttackInfo.statusEffectInfo;

        return new DamageData
        {
            attackerHash    = _ownerHash,
            defenderHash    = defenderHash,
            damage          = elemDamage,
            knockbackForce  = knockback.knockbackForce,
            knockbackDir    = knockback.knockbackDirection,
            physicalType    = _currentAttackInfo.physicalType,
            statusEffectInfo = statusInfo,
            attackType      = _currentAttackInfo.attackType,
            isProjectile    = false,
        };
    }

    /// <summary>
    /// WeaponHolder から攻撃開始時に呼ばれる。
    /// 現在の攻撃情報を設定する。
    /// </summary>
    public void SetAttackInfo(AttackInfo info)
    {
        _currentAttackInfo = info;
    }
}
```

### 3.1 ダメージ計算式

```
【攻撃力計算（装備込み）】

最終攻撃力 = キャラクター基礎攻撃力 + 武器攻撃力 + アクセサリ補正

武器攻撃力の計算（属性ごと）:
  武器攻撃力[e] = weapon.baseAtk[e]
                + scalingCurve.Evaluate(level) × weapon.strScaling  (無属性の場合)
                + scalingCurve.Evaluate(level) × weapon.dexScaling  (無属性の場合)
                + scalingCurve.Evaluate(level) × weapon.intScaling  (魔法属性の場合)

  ※ strScaling/dexScaling は無属性(none)の攻撃力に寄与
  ※ intScaling は火/水/風/土/光/闇の攻撃力に寄与

【基本ダメージ算出（按分計算）】

1. AttackInfo.baseDamage を取得（攻撃固有の基本威力）
2. CharacterAtkStatus.atk（ElementalStatus）から各属性の比率を算出
3. 各属性ダメージ = baseDamage × damageMultiplier × (属性攻撃力 / 全属性攻撃力合計)

例: baseDamage = 100, damageMultiplier = 1.2,
    atk = { none=30, fire=20, 他=0 }
    → none = 120 × 30/50 = 72
    → fire = 120 × 20/50 = 48

【被ダメージ計算（属性別）】

for each element in [None, Fire, Water, Wind, Earth, Light, Dark]:
  ① rawDamage = max(1, atkDamage[e] - defValue[e])
  ② 弱点倍率 = (weakPoint == element) ? 1.5 : 1.0
  ③ 状態異常倍率 = HasEffect(Weakness) ? weaknessModifier : 1.0
  ④ 属性ダメージ = ① × ② × ③
  ※ atkDamage[e] == 0 の属性はスキップ

physicalMod = 1.0 - PhysicalResistance.Get(data.physicalType)
totalDamage = Σ(④) × physicalMod

ガード時: totalDamage = max(1, totalDamage × (1 - GUARD_REDUCTION))

【ガードスタミナコスト】

baseCost = totalDamage × 0.5
shieldReduction = shield.guardEfficiency (0.0〜1.0)
defRatio = clamp(totalDef / totalDamage × 0.3, 0, 1)
finalCost = baseCost × (1 - shieldReduction) × (1 - defRatio)

ガードブレイク条件: currentStamina < finalCost

【パリィ成功条件】

① isParryWindow == true
② attackInfo.isParriable == true
③ data.isProjectile == false

パリィ成功時:
  攻撃者スタン時間 = PARRY_STUN_DURATION (1.5秒)
  反撃ボーナス倍率 = PARRY_COUNTER_MULTIPLIER (1.5倍)
  パリィウィンドウ = weapon.parryWindowDuration + shield.parryWindowBonus
```

### 3.2 ヒットボックス生成

```csharp
/// <summary>
/// 攻撃アニメーションのイベントから呼ばれるヒットボックス。
/// Trigger コライダーで対象を検出し、DamageReceiver に DamageData を送信する。
/// </summary>
public class HitBox : MonoBehaviour
{
    private DamageDealer _dealer;
    private HashSet<int> _alreadyHit; // 同一攻撃で二重ヒット防止

    /// <summary>
    /// ヒットボックスを有効化する。攻撃開始時に呼ばれる。
    /// </summary>
    public void Activate(DamageDealer dealer)
    {
        _dealer = dealer;
        _alreadyHit ??= new HashSet<int>();
        _alreadyHit.Clear();
        gameObject.SetActive(true);
    }

    /// <summary>
    /// ヒットボックスを無効化する。攻撃終了時に呼ばれる。
    /// </summary>
    public void Deactivate()
    {
        gameObject.SetActive(false);
    }

    private void OnTriggerEnter2D(Collider2D other)
    {
        int targetHash = other.gameObject.GetHashCode();

        // 二重ヒット防止
        if (!_alreadyHit.Add(targetHash)) return;

        // GameManager.Data から DamageReceiver 参照を取得
        // （GetComponent ではなく objectHash 経由）
        if (GameManager.Data.TryGetDamageReceiver(targetHash,
                out DamageReceiver receiver))
        {
            DamageData data = _dealer.CalculateOutgoingDamage(targetHash);
            DamageResult result = receiver.ApplyDamage(data);

            // 与ダメージイベントを AI に通知
            if (result.totalDamage > 0)
            {
                AIManager.NotifyEvent(
                    _dealer.OwnerHash,
                    BrainEventFlagType.大ダメージを与えた,
                    targetHash
                );
            }
        }
    }
}
```

### 3.3 飛翔体のダメージ

```csharp
/// <summary>
/// 飛翔体（矢・魔法弾等）が保持するダメージ情報。
/// 生成時に DamageDealer から DamageData を受け取り、
/// 着弾時に DamageReceiver に渡す。
/// </summary>
public class ProjectileDamage : MonoBehaviour
{
    private DamageData _cachedDamage;

    /// <summary>
    /// 飛翔体生成時に DamageDealer から呼ばれる。
    /// ダメージ情報をキャッシュし、isProjectile を true に設定する。
    /// </summary>
    public void SetDamageData(DamageData data)
    {
        data.isProjectile = true;
        _cachedDamage = data;
    }

    private void OnTriggerEnter2D(Collider2D other)
    {
        int targetHash = other.gameObject.GetHashCode();

        // 味方判定（同一陣営には当たらない）
        ref SolidData attackerSolid =
            ref GameManager.Data.GetSolidData(_cachedDamage.attackerHash);
        ref SolidData defenderSolid =
            ref GameManager.Data.GetSolidData(targetHash);

        if (IsSameBelong(attackerSolid, defenderSolid)) return;

        _cachedDamage.defenderHash = targetHash;

        if (GameManager.Data.TryGetDamageReceiver(targetHash,
                out DamageReceiver receiver))
        {
            receiver.ApplyDamage(_cachedDamage);
        }

        // 着弾処理（エフェクト・SE・オブジェクト破棄等）
        OnImpact();
    }

    private void OnImpact()
    {
        // エフェクト再生・オブジェクトプールへ返却
        gameObject.SetActive(false);
    }

    private bool IsSameBelong(in SolidData a, in SolidData b)
    {
        // 陣営（feature）が同一なら味方と判断
        // 具体的な判定ロジックは CharacterFeature のビット演算で行う
        return (a.feature & b.feature) != 0;
    }
}
```

## 4. DamageReceiver（被弾側）

ダメージの受理・軽減・適用を一括管理するコンポーネント。

```csharp
/// <summary>
/// 被弾管理コンポーネント。
/// パリィ判定 → ガード判定 → 属性別ダメージ計算 → HP減算 → 状態異常適用
/// の順でダメージを処理する。
/// データは全て GameManager.Data に objectHash でアクセスする。
/// </summary>
public class DamageReceiver : MonoBehaviour
{
    /// <summary>所有者の objectHash（GameManager.Data へのキー）</summary>
    private int _ownerHash;

    // ---- パリィ関連 ----
    /// <summary>パリィ受付中かどうか</summary>
    private bool _isParryWindow;

    /// <summary>パリィウィンドウの残り時間</summary>
    private float _parryTimer;

    // ---- ガード関連 ----
    /// <summary>ガード中かどうか</summary>
    private bool _isGuarding;

    // ---- 定数 ----
    /// <summary>パリィ成功時の敵スタン時間（秒）</summary>
    private const float PARRY_STUN_DURATION = 1.5f;

    /// <summary>パリィ成功時の反撃ダメージ倍率</summary>
    private const float PARRY_COUNTER_MULTIPLIER = 1.5f;

    /// <summary>弱点属性に対するダメージ倍率</summary>
    private const float WEAKNESS_MULTIPLIER = 1.5f;

    /// <summary>ガード時のダメージ軽減率（1.0 = 100%カット）</summary>
    private const float GUARD_REDUCTION = 0.7f;

    // ===============================================================
    //  初期化
    // ===============================================================

    public void Initialize(int ownerHash)
    {
        _ownerHash = ownerHash;
    }

    // ===============================================================
    //  メインダメージ処理
    // ===============================================================

    /// <summary>
    /// ダメージを適用する。計算順序:
    /// 1. パリィ判定
    /// 2. ガード判定
    /// 3. 属性別ダメージ計算（攻撃力 - 防御力, 弱点倍率）
    /// 4. HP減算
    /// 5. 状態異常判定
    /// 6. 吹き飛ばし適用
    /// 7. AIイベント通知
    /// </summary>
    public DamageResult ApplyDamage(DamageData data)
    {
        var result = new DamageResult();

        // ----- Step 1: パリィ判定 -----
        if (TryParry(data, ref result))
        {
            return result; // パリィ成功 → ダメージ無効
        }

        // ----- Step 2: ガード判定 -----
        bool guarded = TryGuard(data, ref result);

        // ----- Step 3: 属性別ダメージ計算 -----
        int totalDamage = CalculateElementalDamage(data, guarded);
        result.totalDamage = totalDamage;

        // ----- Step 4: HP減算 -----
        ApplyHpReduction(totalDamage, ref result);

        // ----- Step 5: 状態異常判定 -----
        if (!guarded) // ガード中は状態異常無効
        {
            TryApplyStatusEffect(data, ref result);
        }

        // ----- Step 6: 吹き飛ばし適用 -----
        if (!guarded || result.guardBroken)
        {
            ApplyKnockback(data);
        }

        // ----- Step 7: AIイベント通知 -----
        NotifyAI(data, result);

        return result;
    }

    // ===============================================================
    //  パリィ
    // ===============================================================

    /// <summary>
    /// パリィ判定。パリィウィンドウ中かつパリィ可能な攻撃の場合に成功。
    /// </summary>
    private bool TryParry(in DamageData data, ref DamageResult result)
    {
        if (!_isParryWindow) return false;
        if (!IsParriable(data)) return false;

        result.wasParried = true;
        result.totalDamage = 0;

        // 攻撃者にスタンを適用
        ApplyParryStun(data.attackerHash);

        // パリィウィンドウを閉じる
        _isParryWindow = false;
        _parryTimer = 0f;

        return true;
    }

    // ===============================================================
    //  ガード
    // ===============================================================

    /// <summary>
    /// ガード判定。ガード中ならダメージ軽減、スタミナ超過でガードブレイク。
    /// </summary>
    private bool TryGuard(in DamageData data, ref DamageResult result)
    {
        if (!_isGuarding) return false;

        // スタミナ消費計算
        ref StaminaInfo stamina =
            ref GameManager.Data.GetStaminaInfo(_ownerHash);
        float staminaCost = CalculateGuardStaminaCost(data);

        if (stamina.currentStamina < staminaCost)
        {
            // ガードブレイク
            result.guardBroken = true;
            _isGuarding = false;
            stamina.currentStamina = 0;
            return false;
        }

        // スタミナ消費してガード成立
        stamina.currentStamina -= staminaCost;
        result.wasGuarded = true;
        return true;
    }

    // ===============================================================
    //  属性別ダメージ計算
    // ===============================================================

    /// <summary>
    /// 7属性それぞれで (攻撃力 - 防御力) を計算し、弱点倍率・物理耐性を適用する。
    /// </summary>
    private int CalculateElementalDamage(in DamageData data, bool guarded)
    {
        ref CharacterDefStatus defStatus =
            ref GameManager.Data.GetDefStatus(_ownerHash);
        ref SolidData solidData =
            ref GameManager.Data.GetSolidData(_ownerHash);

        int elementalTotal = 0;

        // 各属性のダメージを個別計算（Element enumでループ）
        for (int i = 0; i < 7; i++)
        {
            Element element = (Element)i;
            int atkDmg = data.damage.Get(element);
            if (atkDmg <= 0) continue;

            int defVal = defStatus.def.Get(element);
            elementalTotal += CalcSingleElement(
                atkDmg, defVal, element, solidData.weakPoint);
        }

        // 物理タイプ別耐性を適用
        float physMod = 1.0f - solidData.physicalResistance.Get(data.physicalType);
        int totalDamage = (int)(elementalTotal * physMod);

        // ガード中ならダメージ軽減
        if (guarded)
        {
            totalDamage = Mathf.Max(1, (int)(totalDamage * (1f - GUARD_REDUCTION)));
        }

        return totalDamage;
    }

    /// <summary>
    /// 単一属性のダメージ計算。
    /// finalDamage = max(1, attackDamage - defenderDef)
    /// 弱点属性なら WEAKNESS_MULTIPLIER 倍。
    /// 状態異常 Weakness 中なら追加倍率。
    /// </summary>
    private int CalcSingleElement(
        int atkDamage, int defValue, Element element, Element weakPoint)
    {
        if (atkDamage <= 0) return 0;

        int raw = Mathf.Max(1, atkDamage - defValue);

        // 弱点判定
        if (weakPoint == element)
        {
            raw = (int)(raw * WEAKNESS_MULTIPLIER);
        }

        // 状態異常 Weakness による被ダメージ増加
        ref CharacterStatusEffects effects =
            ref GameManager.Data.GetStatusEffects(_ownerHash);
        if (effects.HasActive(StatusEffectId.Weakness))
        {
            raw = (int)(raw * effects.GetModifier(StatusEffectId.Weakness));
        }

        return raw;
    }

    // ===============================================================
    //  HP減算
    // ===============================================================

    private void ApplyHpReduction(int totalDamage, ref DamageResult result)
    {
        ref CharacterBaseInfo baseInfo =
            ref GameManager.Data.GetBaseInfo(_ownerHash);

        baseInfo.currentHp -= totalDamage;

        if (baseInfo.currentHp <= 0)
        {
            baseInfo.currentHp = 0;
            result.killed = true;
        }

        baseInfo.UpdateRatios();
    }

    // ===============================================================
    //  状態異常
    // ===============================================================

    private void TryApplyStatusEffect(in DamageData data, ref DamageResult result)
    {
        if (data.statusEffectInfo.effect == StatusEffectId.None) return;

        // 蓄積モデル: ヒットごとに蓄積値を加算し、閾値超過で発症
        ref CharacterStatusEffects effects =
            ref GameManager.Data.GetStatusEffects(_ownerHash);

        effects.Accumulate(
            data.statusEffectInfo.effect,
            data.statusEffectInfo.accumulateValue,
            data.statusEffectInfo   // 発症時のパラメータ
        );
        // → 蓄積値 ≥ 閾値 なら内部で自動的に発症スロットに登録される
    }

    // ===============================================================
    //  吹き飛ばし
    // ===============================================================

    private void ApplyKnockback(in DamageData data)
    {
        if (data.knockbackForce <= 0f) return;

        // BaseCharacter の物理挙動に吹き飛ばしを適用
        // GameManager.Data 経由で BaseCharacter 参照を取得
        if (GameManager.Data.TryGetManaged(_ownerHash,
                out BaseCharacter character))
        {
            character.ApplyExternalForce(
                data.knockbackDir * data.knockbackForce
            );
        }
    }

    // ===============================================================
    //  AI通知
    // ===============================================================

    private void NotifyAI(in DamageData data, in DamageResult result)
    {
        // 被弾者 → AIManager に被弾イベントを通知
        AIManager.NotifyEvent(
            _ownerHash,
            BrainEventFlagType.大ダメージを受けた,
            data.attackerHash
        );

        // 死亡時はキル通知
        if (result.killed)
        {
            AIManager.NotifyEvent(
                data.attackerHash,
                BrainEventFlagType.キャラを倒した,
                _ownerHash
            );
        }
    }

    // ===============================================================
    //  パリィ補助メソッド
    // ===============================================================

    /// <summary>
    /// パリィウィンドウを開始する。ガード入力直後に呼ばれる。
    /// </summary>
    public void StartParryWindow(float duration)
    {
        _isParryWindow = true;
        _parryTimer = duration;
    }

    /// <summary>ガード状態を設定する。</summary>
    public void SetGuarding(bool guarding)
    {
        _isGuarding = guarding;
    }

    /// <summary>
    /// パリィ可能な攻撃かどうかを判定する。
    /// </summary>
    private bool IsParriable(in DamageData data)
    {
        // 環境ダメージはパリィ不可
        if (data.attackType == AttackType.Environmental) return false;
        // 飛翔体はパリィ不可（設定で変更可能にする場合はここを拡張）
        if (data.isProjectile) return false;
        return true;
    }

    /// <summary>
    /// パリィ成功時に攻撃者にスタンを適用する。
    /// </summary>
    private void ApplyParryStun(int attackerHash)
    {
        // 攻撃者にスタン状態異常を即時発症させる
        ref CharacterStatusEffects attackerEffects =
            ref GameManager.Data.GetStatusEffects(attackerHash);
        attackerEffects.ForceApply(
            StatusEffectId.Stun,
            PARRY_STUN_DURATION
        );
    }

    private void Update()
    {
        // パリィウィンドウのタイマー管理
        if (_isParryWindow)
        {
            _parryTimer -= Time.deltaTime;
            if (_parryTimer <= 0f)
            {
                _isParryWindow = false;
            }
        }
    }

    /// <summary>
    /// ガード時のスタミナ消費量を計算する。
    /// 盾のガード効率と防御力で軽減される。
    /// </summary>
    private float CalculateGuardStaminaCost(in DamageData data)
    {
        // 基本コスト = 合計ダメージの一定割合
        float baseCost = data.damage.Total * 0.5f;

        // 盾のガード効率による軽減
        ref EquipmentStatus equipStatus =
            ref GameManager.Data.GetEquipmentStatus(_ownerHash);
        float shieldReduction = equipStatus.guardEfficiency;

        // 防御力による軽減
        ref CharacterDefStatus defStatus =
            ref GameManager.Data.GetDefStatus(_ownerHash);
        float defRatio = Mathf.Clamp01(
            (float)defStatus.def.ReturnSum() /
            Mathf.Max(1, data.damage.Total) * 0.3f);

        return baseCost * (1f - shieldReduction) * (1f - defRatio);
    }
}
```

### 6.2 パリィ成功時の効果

パリィが成功すると以下の3つの効果が発生する。

```
パリィ成功
    │
    ├─ ① ダメージ完全無効
    │     result.totalDamage = 0
    │     result.wasParried = true
    │
    ├─ ② 攻撃者にスタンを適用
    │     StatusEffectId.Stun を ForceApply（蓄積を経ずに即発症）
    │     持続時間: PARRY_STUN_DURATION (1.5秒)
    │     → スタン中は行動不能
    │
    └─ ③ 反撃ボーナス
          次の攻撃のダメージ倍率: PARRY_COUNTER_MULTIPLIER (1.5倍)
          持続時間: スタン時間と同等
          → DamageDealer 側で倍率を参照
```

```csharp
/// <summary>
/// パリィ成功時の処理まとめ。
/// </summary>
private void OnParrySuccess(in DamageData data, ref DamageResult result)
{
    // ① ダメージ無効
    result.wasParried = true;
    result.totalDamage = 0;

    // ② 攻撃者スタン
    ApplyParryStun(data.attackerHash);

    // ③ 反撃ボーナス（CharacterStateInfo にフラグ設定）
    ref CharacterStateInfo selfState =
        ref GameManager.Data.GetStateInfo(_ownerHash);
    // パリィ成功フラグを設定 → DamageDealer が次の攻撃時に参照
    selfState.brainEvent |= BrainEventFlagType.攻撃対象指定;

    // パリィエフェクト・SE 再生
    // EffectManager.Play(ParryEffect, transform.position);
}
```

|---|---|---|

| スキル攻撃（Skill） | **可能** | 強力だがパリィで対処可能 |
| 魔法攻撃（Magic） | **不可** | 魔法はガードのみ |

| 飛翔体（isProjectile = true） | **不可** | 矢・魔法弾等はパリィ不可 |
| ボス専用攻撃（将来拡張） | **設定次第** | AttackInfo にパリィ可否フラグを追加予定 |

## 7. ガードシステム

### 7.1 ガード中のダメージ軽減率

```
ガード中のダメージ計算:

  通常ダメージ = Σ max(1, atkDamage[e] - def[e]) × 弱点倍率
  ガード後ダメージ = 通常ダメージ × (1 - GUARD_REDUCTION)
                   = 通常ダメージ × 0.3
```

| 項目 | 値 | 説明 |
|---|---|---|
| 基本軽減率 | 70% | `GUARD_REDUCTION = 0.7f` |
| 最低ダメージ | 1 | ガード中でも最低1ダメージは受ける |
| 状態異常 | 無効 | ガード成立中は状態異常を完全ブロック |
| 吹き飛ばし | 無効 | ガード成立中は吹き飛ばし無効 |
| スタミナ消費 | あり | ダメージ量に応じたスタミナを消費 |

### 7.2 ガードブレイク（スタミナ超過時）

```
ガードブレイク判定:

  ガードスタミナコスト = ダメージ合計 × 0.5 × (1 - 防御補正)

  if (currentStamina < ガードスタミナコスト):
      ガードブレイク発生
      ├─ ガード解除 (_isGuarding = false)
      ├─ スタミナ = 0
      ├─ ダメージ軽減なし（フルダメージ適用）
      ├─ 吹き飛ばし適用
      └─ スタン状態（短時間行動不能）
```

### 7.3 属性別ガード耐性

ガード時のスタミナ消費は**盾のガード効率**と**防御力**で軽減される。

```
ガードスタミナコスト計算:

  baseCost = ダメージ合計 × 0.5
  盾軽減 = shield.guardEfficiency (0.0〜1.0)
  防御補正 = clamp(防御力合計 / ダメージ合計 × 0.3, 0, 1)
  最終コスト = baseCost × (1 - 盾軽減) × (1 - 防御補正)

  例: ダメージ合計 = 100, guardEfficiency = 0.4, 防御力合計 = 50
      baseCost = 50
      盾軽減後 = 50 × (1 - 0.4) = 30
      防御補正 = clamp(50/100 × 0.3, 0, 1) = 0.15
      最終コスト = 30 × 0.85 = 25.5
```

| 盾効率 | 防御力/ダメージ比 | 最終コスト(ダメージ100時) |
|---|---|---|
| 0.0 | 0% | 50.0 |
| 0.4 | 0% | 30.0 |
| 0.4 | 50% | 25.5 |
| 0.7 | 100% | 10.5 |
| 1.0 | 任意 | 0（完全防御） |

## 8. 状態異常管理システム

### 8.1 設計方針

従来のビットフラグ + 確率判定方式を廃止し、**蓄積→発症モデル**を採用する。
攻撃ヒットごとに蓄積値を加算し、閾値を超えた時点で状態異常が発症する。
これにより、ボスや高耐性キャラへの段階的な状態異常適用や、
連続攻撃による状態異常の確実な蓄積が実現できる。

### 8.2 StatusEffectId（状態異常ID）

旧 SpecialEffect enum を廃止。

```csharp
    Bleed,        // 出血: 移動時に追加ダメージ
```

### 8.4 AttackInfo 用の StatusEffectInfo

```csharp
/// <summary>
/// 攻撃に付随する状態異常情報。AttackInfo に組み込む。

{
    public StatusEffectId effect;           // 付与する状態異常
    public float accumulateValue;           // 1ヒットあたりの蓄積量
    public float duration;                  // 発症時の持続時間(秒)
    public float tickDamage;               // tick毎のダメージ量(DoT用)
    public float tickInterval;             // tickの間隔(秒, DoT用)
    public float modifier;                 // 効果量(速度低下率、被ダメ増加率等)
    public byte maxStack;                  // 最大スタック数(0=スタック不可)
}
```

### 8.5 蓄積→発症フロー

```
攻撃ヒット (StatusEffectInfo付き)
    │
    ▼
┌─────────────────────────────────────────────────┐
│ 蓄積フェーズ                                    │
│                                                  │
│  耐性補正後の蓄積量:                             │
│    adjustedValue = accumulateValue                │
│                  × (1 - statusResistance)        │
│                                                  │
│  蓄積スロットを検索（同一IDまたは空きスロット）  │
│  accumulation += adjustedValue                   │
│                                                  │
│  if (accumulation >= threshold):                 │
│    → 発症: StatusEffectSlot に登録               │
│    → accumulation = 0 にリセット                 │
│    → 既に発症中 && stackable:                    │
│        currentStack++ (maxStack上限)             │
│        remainingDuration をリセット               │
│    → activeFlags に該当ビットをセット            │
└─────────────────┬───────────────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────────────┐
│ 毎フレーム更新 (StatusEffectManager.Update)     │
│                                                  │
│  for each activeSlot:                            │
│    remainingDuration -= deltaTime                │
│    if (remainingDuration <= 0):                  │
│      → 解除: slot.id = None                     │
│      → activeFlags から該当ビットをクリア        │
│      → continue                                 │
│                                                  │
│    tickTimer -= deltaTime                        │
│    if (tickTimer <= 0):                          │
│      → DoT処理: HP -= damagePerTick × stack     │
│      → tickTimer += tickInterval                │
│                                                  │
│  for each accumSlot:                             │
│    accumulation -= decayRate × deltaTime         │
│    accumulation = max(0, accumulation)           │
└─────────────────────────────────────────────────┘
```

### 8.7 StatusEffectManager

```csharp
/// <summary>
/// 全キャラクターの状態異常を毎フレーム更新するマネージャー。
/// GameManager の子コンポーネント。SoAコンテナを直接操作する。
/// </summary>
public class StatusEffectManager : MonoBehaviour
{
    public void UpdateAllEffects(float deltaTime)
    {
        var data = GameManager.Data;
        for (int i = 0; i < data.Count; i++)
        {
            ref CharacterStatusEffects effects = ref data.GetStatusEffectsAt(i);
            if (effects.activeFlags == 0 &&
                effects.accum0.targetEffect == StatusEffectId.None &&
                effects.accum1.targetEffect == StatusEffectId.None &&
                effects.accum2.targetEffect == StatusEffectId.None &&
                effects.accum3.targetEffect == StatusEffectId.None)
                continue; // 処理不要

            // 発症中スロットの更新
            UpdateSlot(ref effects.slot0, ref effects, i, deltaTime);
            UpdateSlot(ref effects.slot1, ref effects, i, deltaTime);
            UpdateSlot(ref effects.slot2, ref effects, i, deltaTime);
            UpdateSlot(ref effects.slot3, ref effects, i, deltaTime);

            // 蓄積値の自然減衰
            DecayAccumulation(ref effects.accum0, deltaTime);
            DecayAccumulation(ref effects.accum1, deltaTime);
            DecayAccumulation(ref effects.accum2, deltaTime);
            DecayAccumulation(ref effects.accum3, deltaTime);
        }
    }

    private void UpdateSlot(ref StatusEffectSlot slot,
        ref CharacterStatusEffects effects, int charIndex, float dt)
    {
        if (slot.id == StatusEffectId.None) return;

        slot.remainingDuration -= dt;
        if (slot.remainingDuration <= 0f)
        {
            effects.activeFlags &= (ushort)~(1 << (int)slot.id);
            slot.id = StatusEffectId.None;
            return;
        }

        // DoT処理（tickInterval > 0 の場合のみ）
        if (slot.tickInterval > 0f)
        {
            slot.tickTimer -= dt;
            if (slot.tickTimer <= 0f)
            {
                float damage = slot.damagePerTick * slot.currentStack;
                ref CharacterBaseInfo baseInfo =
                    ref GameManager.Data.GetBaseInfoAt(charIndex);
                baseInfo.currentHp -= (int)damage;
                slot.tickTimer += slot.tickInterval;
            }
        }
    }

    private void DecayAccumulation(ref StatusAccumulation accum, float dt)
    {
        if (accum.targetEffect == StatusEffectId.None) return;
        accum.accumulation = Mathf.Max(0f,
            accum.accumulation - accum.decayRate * dt);
        if (accum.accumulation <= 0f)
            accum.targetEffect = StatusEffectId.None; // スロット解放
    }
}
```

## 9. DamageResult

ダメージ処理の結果を呼び出し元に返す構造体。

```csharp
/// <summary>
/// DamageReceiver.ApplyDamage の戻り値。
/// 攻撃側（DamageDealer / HitBox）がこの結果を受けて
/// エフェクト表示、AI通知、コンボ継続判定などを行う。
/// </summary>
public struct DamageResult
{
    /// <summary>実際に適用された合計ダメージ量</summary>
    public int totalDamage;

    /// <summary>パリィで無効化されたかどうか</summary>
    public bool wasParried;

    /// <summary>ガードで軽減されたかどうか</summary>
    public bool wasGuarded;

    /// <summary>ガードブレイクが発生したかどうか</summary>
    public bool guardBroken;

    /// <summary>この攻撃で対象が死亡したかどうか</summary>
    public bool killed;

    /// <summary>蓄積により発症した状態異常（発症なしなら StatusEffectId.None）</summary>
    public StatusEffectId triggeredEffect;
}
```

### 9.1 DamageResult の使用例

```csharp
// HitBox 内での使用例
DamageResult result = receiver.ApplyDamage(data);

if (result.wasParried)
{
    // パリィエフェクト再生
    EffectManager.Play(EffectType.Parry, hitPoint);
    SoundManager.Play(SoundType.ParrySuccess);
}
else if (result.wasGuarded)
{
    // ガードエフェクト再生
    EffectManager.Play(EffectType.Guard, hitPoint);
    SoundManager.Play(SoundType.GuardHit);

    if (result.guardBroken)
    {
        // ガードブレイクエフェクト
        EffectManager.Play(EffectType.GuardBreak, hitPoint);
        SoundManager.Play(SoundType.GuardBreak);
    }
}
else
{
    // 通常ヒットエフェクト
    EffectManager.Play(EffectType.Hit, hitPoint);
    SoundManager.Play(SoundType.Hit);

    if (result.killed)
    {
        // 撃破エフェクト
        EffectManager.Play(EffectType.Kill, hitPoint);
    }
}

// ダメージ数値表示
if (result.totalDamage > 0)
{
    DamageNumberUI.Show(hitPoint, result.totalDamage,
        result.triggeredEffect != StatusEffectId.None);
}
```

### 10.2 AIイベントの活用

通知された `BrainEventFlagType` は `CharacterStateInfo.brainEvent` に蓄積され、次回の AI判定時に AI 判定で参照される。

| イベント | AI側の反応例 |
|---|---|
| `大ダメージを受けた` | 回避行動に移行 / 回復アビリティ使用 / モード変更（攻撃→防御） |
| `大ダメージを与えた` | 追撃判断 / コンボ継続 |
| `キャラを倒した` | ターゲット変更 / 新たな敵を探索 |

```csharp
private void UpdateRecognitionOnHit(in DamageData data)
{
    ref RecognitionData recognition =
        ref GameManager.Data.GetRecognitionData(_ownerHash);

    // 最後に攻撃してきた相手を記録
    recognition.attackerHash = data.attackerHash;
}
```

## 12. データアクセスまとめ

本システムの全てのデータアクセスは `GameManager.Data`（SoACharaDataDic）経由で行い、`GetComponent` は使用しない。

| アクセス対象 | メソッド | 使用場面 |
|---|---|---|
| HP / MP | `GetBaseInfo(hash)` | HP減算、死亡判定 |
| 攻撃力（7属性） | `GetAtkStatus(hash)` | ダメージ算出 |
| 防御力（7属性） | `GetDefStatus(hash)` | ダメージ軽減計算 |
| 弱点属性 / 陣営 | `GetSolidData(hash)` | 弱点倍率、味方判定 |
| 状態異常 / イベント | `GetStateInfo(hash)` | 状態異常適用、AI通知 |
| スタミナ | `GetStaminaInfo(hash)` | ガードスタミナ消費 |
| 認識データ | `GetRecognitionData(hash)` | ヘイト更新 |
| BaseCharacter 参照 | `TryGetManaged(hash)` | 吹き飛ばし適用 |
| DamageReceiver 参照 | `TryGetDamageReceiver(hash)` | ヒット時のダメージ送信 |
