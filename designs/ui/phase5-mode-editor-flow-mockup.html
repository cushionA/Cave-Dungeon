Phase 5: 仲間AI設定 — モード詳細 ▶ 行動詳細 ▶ 個別エディタ

モード詳細ウインドウの行動リストから任意の行をクリックすると、その行動と条件が並んだ 行動詳細ウインドウが開く。詳細ウインドウ内で行動ラベルや各条件をクリックすると、 それぞれ個別エディタが開いて中身を編集できる。
デフォルト行動は「何もしない」で固定（ルール不一致時のアイドル状態）。クリック不可。

設計メモ

  • ステップ1（モード詳細）: 1行=1ルールで表示。[優先度] 行動名 ─ 条件サマリ の並び。行動をプライマリに視覚化、条件は補助。
  • ステップ2（行動詳細）: 行をクリックすると開く。行動カード（1枚）+ 条件リスト（複数）+ 発動確率が同居。フッタに「別条件で複製」「削除」を配置して、中間メニュー層を省略。
  • ステップ3A（ActionPicker）: 行動ラベルをクリックすると開く。タブ切替で全行動タイプから選択。
  • ステップ3B（条件エディタ）: 条件行をクリックすると開く。条件種類・演算子・閾値・フィルタをまとめて編集。下部に自然文プレビュー。
  • デフォルト行動: 何もしない で固定。モード詳細のリスト末尾に DEFAULT バッジ付きで表示するが、クリック不可・編集不可。
  • データモデル整合: 内部は従来通り actions[] + actionRules[]（actionIndexで参照）を維持。「別条件で複製」は actionIndexを共有する形で実装され、UI上は見えない。保存時に未参照スロットを GC。
